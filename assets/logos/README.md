# Logo Assets

These files are used only to identify customer and program context on the EOS public website.

- `eos-header-logo.svg`: Header logo converted from `260430 eos LOGO final.pdf`.
- `eos-footer-logo.svg`: Footer logo converted from `ELEVATE LOGO 320.pdf`.
- `favicon.svg`: Browser favicon extracted from the symbol portion of `eos-header-logo.svg`.
- `hyundai-motor.svg`: Hyundai Motor Company logo downloaded from Wikimedia Commons via `https://commons.wikimedia.org/wiki/Special:Redirect/file/Hyundai_Motor_Company_logo.svg`
- `kia.svg`: Kia logo downloaded from Wikimedia Commons via `https://commons.wikimedia.org/wiki/Special:Redirect/file/KIA_logo3.svg`

All trademarks and brand assets remain the property of their respective owners.
